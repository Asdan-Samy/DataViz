Mentions légales

Bienvenue sur la page des mentions légales du site DataCollege. Nous tenons à garantir la transparence sur les données utilisées et à respecter les droits des utilisateurs. Retrouvez ici toutes les informations légales concernant ce projet.

Sources et Références – DataCollege

Les données utilisées dans ce projet proviennent de diverses sources fiables, notamment :

  • Source 1 : Classement de Shanghai (ARWU)
    Description : Le Classement de Shanghai classe les universités mondiales en fonction de critères académiques, tels que le nombre de publications scientifiques, les prix Nobel et les performances en recherche.
    Lien vers la source

POLICES UTILISÉES

Les polices suivantes ont été utilisées dans le projet pour garantir une lisibilité optimale et une expérience utilisateur agréable :

  • SeoulNamsan CBL – Une police élégante et moderne pour le contenu principal.
  • Sevillana – Une police stylisée pour les titres secondaires.
  • Shadows Into Light – Une police manuscrite pour des éléments décoratifs.

NOM LIBRE DE DROITS

Le nom DataCollege a été vérifié pour garantir qu'il est libre de droits et qu'il n'est pas déjà enregistré. Une recherche approfondie a été effectuée sur la base de données officielle de l'INPI.

Vous pouvez consulter la recherche en visitant le lien suivant : Recherche INPI - DataCollege
